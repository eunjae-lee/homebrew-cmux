# homebrew-cmux

Homebrew tap for [cmux fork](https://github.com/eunjae-lee/cmux) with workspace provider support.

## Install

```bash
brew tap eunjae-lee/cmux
brew install --cask cmux-fork
```

## Update

```bash
brew upgrade --cask cmux-fork
```

## Note

This is an unsigned build. On first launch, right-click the app → Open to bypass Gatekeeper.
